# TypeScript Master Class (Backend Focus)

This repository contains a 7-day master class for advanced TypeScript, specifically tailored for backend development.
Managed as a Bun monorepo.

## Curriculum Roadmap

- **[Day 1: Advanced Types & Runtime Foundation](./day1-advanced-types-runtime)**
  - Conditional Types, Mapped Types, Runtime Validation (Zod), Branded Types.

- **[Day 2: Functional Core & Error Handling](./day2-functional-patterns)**
  - Algebraic Data Types (ADTs), Result Pattern vs Exceptions, Pipe/Compose, Immutability.

- **[Day 3: Async Mastery & Concurrency](./day3-async-concurrency)**
  - Async Iterators, Generators, AbortController, Structured Concurrency, Task Queues.

- **[Day 4: Metaprogramming & Decorators](./day4-metaprogramming)**
  - Reflection API, Method/Class Decorators, Metadata, Building a mini-DI container.

- **[Day 5: High-Performance IO & Streams](./day5-performance-streams)**
  - Node/Bun Streams, Buffers, Backpressure handling, parsing large datasets efficiently.

- **[Day 6: Enterprise Architecture & Patterns](./day6-architecture)**
  - Dependency Injection (Inversify/Custom), Clean Architecture, Domain Driven Design (DDD) in TS.

- **[Day 7: The Compiler API & AST](./day7-compiler-api)**
  - Manipulating code with code. Abstract Syntax Trees, custom transformers, and static analysis tools.

## Prerequisites
- Basic to Intermediate TypeScript knowledge.
- NodeJS / Bun installed.

## Setup
```bash
bun install
```
