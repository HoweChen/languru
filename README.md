# 🚀 Languru: The Polyglot Engineering Master Class

> **Level Up Your Engineering Craft. No Basics. Pure Advanced Patterns.**

Welcome to **Languru**, a curated collection of advanced engineering master classes designed to take experienced developers from "proficient" to "expert" in modern backend languages. 

This repository avoids "Hello World" tutorials. Instead, we focus on **internals, architecture, performance, and distributed patterns**—the knowledge that distinguishes senior engineers.

---

## 📚 Curriculum Tracks

### 🔷 [Track 1: Golang System Mastery](./golang_master_class)
*7 Days to Conquer High-Performance Systems Engineering*

Go beyond simple CRUD. This track explores the runtime, memory model, and distributed system patterns that power cloud-native infrastructure.

- **Day 1: Modern Foundations** — Structured Concurrency, New Iterators, and Runtime Behaviors.
- **Day 2: Memory & Generics** — Escape Analysis, GC Tuning, and Advanced Generic Patterns.
- **Day 3: Metaprogramming** — Building Custom DI Containers & ORMs with Reflection.
- **Day 4: Network & RPC** — gRPC Interceptors, HTTP Middleware Design, and Protocol Buffers.
- **Day 5: Extreme Performance** — pprof Profiling, Assembly Analysis, and Optimization Techniques.
- **Day 6: Distributed Patterns** — Distributed Locking, Circuit Breakers, and Consistency Models.
- **Day 7: Architecture & Unsafe** — `unsafe` Pointer Magic and Clean Architecture Implementation.

### 🔷 [Track 2: TypeScript Backend Architect](./typescript_master_class)
*7 Days to Master Enterprise Node.js & Bun Development*

TypeScript is more than just types. This track focuses on metaprogramming, functional patterns, and compiler APIs for building robust enterprise backends.

- **Day 1: Type System Wizardry** — Conditional/Mapped Types, Zod Runtime Validation, and Branded Types.
- **Day 2: Functional Core** — Algebraic Data Types (ADTs), Result Patterns, and Immutability.
- **Day 3: Async Mastery** — Event Loop Internals, Structured Concurrency, and Task Queues.
- **Day 4: Metaprogramming** — Decorators, Reflection API, and Metadata Injection.
- **Day 5: High-Performance I/O** — Stream Processing, Backpressure, and Buffer Manipulation.
- **Day 6: Enterprise Patterns** — Dependency Injection (Inversify), DDD, and Hexagonal Architecture.
- **Day 7: Compiler API** — AST Manipulation, Custom Transformers, and Static Analysis Tools.

## 🎯 Philosophy

1.  **Depth over Breadth**: We drill deep into language internals.
2.  **Zero Fluff**: No basic syntax explanations. We assume you know how to code.
3.  **Pattern-First**: Focus on reusable architectural patterns (DI, Repository, Middleware).
4.  **Polyglot Mindset**: Compare how different runtimes handle concurrency and memory.

## 🛠️ Getting Started

Each track is self-contained with its own examples and exercises.

```bash
# Clone the repository
git clone https://github.com/yourusername/languru.git

# Explore TypeScript Track

```
